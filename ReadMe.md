# PathTracer_POSIX
A multi-threaded path tracer built in C++ and PThreads. Based on [Kevin Beason's smallpt](https://drive.google.com/file/d/0B8g97JkuSSBwUENiWTJXeGtTOHFmSm51UC01YWtCZw/view).

To compile :
```
g++ -pthread path_tracer.cpp 
```
